# Zhannah Portfolio — Deploy Guide

## 🚀 How to get it live for FREE (custom domain optional)

### Option 1 — Netlify (easiest, recommended)
1. Go to **netlify.com** → sign up free
2. Drag and drop the `zhannah-portfolio-onepage` folder onto the Netlify dashboard
3. Your site is live instantly at a URL like `zhannah.netlify.app`
4. **Free custom domain**: In Netlify → Site settings → Domain management → Add custom domain
   - Get a free `.netlify.app` subdomain (e.g. `zhannah.netlify.app`)
   - Or buy a cheap domain (~£10/yr on Namecheap) and connect it

### Option 2 — GitHub Pages (also free)
1. Create a free account at **github.com**
2. New repo → upload all files → Settings → Pages → Deploy from main branch
3. Live at `yourusername.github.io/reponame`

### Option 3 — Vercel (free, fastest CDN)
1. Go to **vercel.com** → sign up free
2. Import from GitHub or drag & drop
3. Live at `zhannah.vercel.app`

## 📝 To update your work
Edit `js/data.js` — that's the only file you need to change to add/remove films.
